# datagate-datacenter
Data center access control system with real-time monitoring and request management

## Overview

DataGate is a web system for managing access to data centers.
It replaces manual tools like Google Forms with a secure and automated solution.

---

## Features

* Access request form
* Automatic protocol generation
* Real-time dashboard (Socket.io)
* Company and rack management
* Equipment entry/exit control
* Email notifications

---

## Technologies

* Node.js
* Express
* SQLite
* Socket.io
* EJS
* Nodemailer

---

## How to Run

```bash
npm install
node Acess.js
```

Access:

* http://localhost:3000
* http://localhost:3000/admin

---

## Purpose

This project was developed to improve control, security, and organization of access in data center environments.
